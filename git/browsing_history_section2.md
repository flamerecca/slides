## See The World
<hr>

```bash
~$ git log --all --date-order --decorate --graph --oneline --simplify-by-decoration

--all                        Pretend as if all the refs in refs/.
--date-order                 Show commits in the commit timestamp order.
--decorate                   Print out the ref names [=short|full|auto|no]. 
--graph                      Draw a text-based graphical representation of the commit history.
--oneline                    A shorthand for "--pretty=oneline --abbrev-commit" used together.
--simplify-by-decoration     Commits that are referred by some branch or tag are selected.
```

[Git Log Docs](https://git-scm.com/docs/git-log)
